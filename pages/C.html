  • Compilando meu 1º código? O que é o GCC? Comando explicado
    • GNU Compiler Collection ou Coleção de compilação do GNU é um conjundo de compiladores de linguagem de programação, ou seja ele ira pegar o nosso código que é legivel por humanos e o tranformará em binário, linguagem de máquina. O comando de uso do GCC é: 
    gcc nomeDoSeuArquivo.c -o arquivoFinal
    -o : refere-se ao output (Saída do programa)
